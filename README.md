## What I Build

I write game systems for Roblox: the kind that ship, scale, and hold up under real player load. My work spans the full stack: game logic, networking, UI, persistence, and NPC behavior. I care about architecture, not just getting it to run.

**Core focus areas:**

- **Game Systems:** combat, guns, tycoons, tower defense, movement, progression
- **AI & NPCs:** pathfinding, boss behaviors, state machines, zombie wave logic
- **DataStore & Persistence:** player saves, house/inventory systems, anti-data-loss
- **UI/UX:** clean, responsive interfaces using ScreenGui best practices
- **Networking:** RemoteEvent/RemoteFunction architecture, server authority, anti-exploit patterns
